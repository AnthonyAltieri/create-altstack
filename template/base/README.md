# {{PROJECT_NAME}}

A monorepo bootstrapped with [create-altstack](https://github.com/AnthonyAltieri/alt-stack).

## Getting Started

```bash
# Install dependencies
pnpm install

# Start development
pnpm dev

# Build all apps and packages
pnpm build

# Lint the codebase
pnpm lint
```

## Project Structure

```
├── apps/           # Application packages
├── packages/       # Shared packages
├── turbo.json      # Turborepo configuration
└── pnpm-workspace.yaml
```
